📦 Quick Configuration Presets

Use the buttons below to quickly configure common OER sources with pre-configured settings. Verified presets are marked with ✅
🌐 API Sources

REST API-based OER sources – good for fresh data and incremental updates. Most flexible but metadata quality varies.

📚 OAI-PMH Sources

OAI-PMH protocol repositories – standardized Dublin Core metadata. Best for library/repository interoperability.

⚠️ Note: OAPEN and DOAB OAI-PMH endpoints are currently broken. Use KBART or MARCXML presets instead (see below).

📊 CSV/TSV/KBART Sources

CSV/KBART file sources – simple curated lists. Good for one-time bulk imports or publisher catalogs.

⭐ Recommended: Use OAPEN KBART for reliable OAPEN harvesting (OAI-PMH endpoint is broken).

📜 MARCXML / Dump Sources

MARCXML dumps – richest bibliographic metadata with full MARC21 fields. Best for detailed cataloging.
